Admin Commands

Moderation commands
!ckick @user reason - Kicks the mentioned user from the server.
!cban @user reason - Bans the mentioned user from the server.
!cunban userID - Unbans the user if given their user ID.
!cwarn - Warns the mentioned user. Example: !cwarn @user reason
Utility commands !cannounce - Announces if there is a new announcement.
!crolecolor Hex color code -Changes Mentioned role color, if specified with a hex color code. Example: !crolecolor @Role #00FFFF
!cnews - Announces if there's a news!
!cname @user - (Admins only, if left blank after !cname @user,the nickname clears.The bot roles must be above all other roles for this to work.) Changes the nickname of the mentioned user!
!cclear (number of messages) - clears messages.
!cexc - Clears exact 70 messages.
!csetwelcome #channel - Sets the welcome message to the mentioned channel (Currently outaged).
!cDM @user - DMs the mentioned user.
!cpoll question - Creates a poll in the channel where you send this message.
!clock - Locks the channel
!cunlock - Unlocks the channel
!csetreminder #channel - Sets the reminder to the mentioned channel, Cosmic will post messages in that channel whenever there is a new announcement in the announcement channel, begin the announcement with Hey @everyone or @everyone.
!csetlogs #channel - Set the logs to the mentioned channel.
!csetannounce - Set the announcement channel, this would be required for the reminding messages of Cosmic.
!cadminsay text - Cosmic makes a formal message in the channel you are executing the command in.
